### 命令
```
/lobby open <game_name>     开启游戏大厅
/lobby close <game_name>    关闭游戏大厅
/lobby finish <game_name>   强制结束游戏
/lobby join <game_name>     加入游戏大厅
/lobby quit                 退出游戏大厅
/lobby list                 列出已注册游戏大厅
```
